# Rocq

Formally verified theorems in [Rocq](https://rocq-prover.org/).

## Building

Requires [Rocq 9.1](https://rocq-prover.org/). If you have it installed, just run:

```
make
```

Alternatively, open this repo in a [dev container](https://containers.dev/) (e.g. VS Code or GitHub Codespaces) — everything is pre-configured.
